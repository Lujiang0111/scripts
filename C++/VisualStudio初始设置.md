# Visual Studio初始设置

## vs2022自定义设置

### 工具->选项

1. 环境->字体和颜色：
    + 字体：Consolas
    + c++变量：**185，119，30**
    + c++参数：**156，220，254**
    + C++成员函数、C++函数、C++函数模板、C++静态成员函数：**227，227，164**
    + c++枚举器：**150，215，163**
    + c++命名空间：**184，215，163**
    + c++字段：**189，183，107**

2. 文本编辑器->C/C++
    + 制表符：选择**插入空格**
    + 代码样式->格式设置->间距->指针/引用对齐方式：选择**右对齐**
